## companywebsite
基于**Laravel5.3** 开发

## 安装

### 1.克隆源码到本地
> git clone http://54ndd.com:88/cholding.git

### 2.进入项目目录
> cd cholding

### 3. 拷贝`.env`文件
> cp .env.example .env

### 4. 下载相关的依赖包(也可以使用yarn)
下载`laravel`相关依赖的包
> composer install

### 5. 创建数据
> php artisan changhe:install

### 6.管理员创建
> php artisan create:admin
